`Tlumx\log` - component for logging, that implement the PSR-3 Logger Interface.

## Requirements

* PHP >= 7.0

## Installation

Use [Composer](https://getcomposer.org/) to install tlumx-log.

```bash
$ composer require tlumx/tlumx-log
```
